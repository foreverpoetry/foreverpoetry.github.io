hexo设置 关于 标签 分类 归档

发表于 2018-04-30 | 分类于 hexo |

在搭建起hexo后，我使用next做为主题，以下是关于怎么添加关于、标签、分类、归档的方法

1：添加 关于页面

使用：hexo new page "about" 新建一个 关于我 页面,在创建的同时也会创建一个about的文件夹，里面的index.md文件就是显示的页面，在里面添上你的介绍。如果使用了评论插件，可以设置这个页面不评论comments: false。

然后在next主题的 _config.yml 文件中的 menu 中进行匹配

2：添加 标签页面

同样使用： hexo new page tags 新建一个 标签 页面,重点是type: "tags",最好也不要评论。

然后再在主题的 _config.yml 文件中的 menu 中进行匹配

然后在新建hexo new "new md"的文章中加入tags标签：

tags 也可以写成tags: [tags 1,tags 2]

3：添加 分类页面

添加分类页面和tags页面比较类似。使用： hexo new page "categories" 新建一个 分类 页面。
主题的 _config.yml 文件中的 menu 中进行匹配

4：添加 归档页面

主题的 _config.yml 文件中的 menu 中进行匹配

5：添加 自定义页面

以上是添加特定页面的方法，其实都是使用hexo new page创建一个新的页面，然后给特定的页面以特殊功能。添加自定义页面也就是： hexo new page "newpage" 新建一个 自定义 页面。
在主题的 _config.yml 文件中的 menu 中添加一行，标识增加一个自定义页面

# hexo # new page
